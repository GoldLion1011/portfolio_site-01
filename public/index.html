Intro

A near-native of Tulsa, OK. By the way, check out my awesome work.

As a junior Fullstack Web Developer, I have developed a passion for frontend development and UX/UI design. I enjoy experimenting with and creating intuitive and visually appealing user experiences that bring websites to life. My strong foundation in web technologies and keen design sense enable me to combine technical skills with creative problem-solving to build dynamic and user-friendly applications. I constantly explore new frontend frameworks, stay up-to-date with the latest industry trends, and hone my skills to deliver high-quality web solutions. With a background in healthcare, wellness, and the visual arts, I have worked on various projects that involve building responsive and accessible websites. I strongly believe in the power of collaboration and strive to create seamless experiences that engage and delight users.

Works

  • Tulsa Air and Space Museum Visitor Portal

    In partnership with TASM, we are developing an interactive visitor sign-in portal as our capstone project for Holberton Tulsa. More details to come!

  • DraCo

    Our team was given the task to create a website for a confidential client using the prompt "Draco dormiens nunquam titillandus". We decided to embrace the theme and created a luxury boutique for dragons, with a Firebase backend. Our tech stack consisted of HTML, JavaScript, Firebase, Firestore, Canva, and Bootstrap.

  • The Joy of Painting API

    I worked on a solo project where I parsed files, set up a server, and created a SQL database that provides a list of "Joy of Painting" episodes based on specific criteria. It was a lot of fun to work on and I'm happy with how it turned out!

  • NaviSavi - Weather Data Historical Integration Demo

    In the works! Group project for NaviSavi, the video-only travel app.

  • AirBnB Clone

    An AirBnB clone was created with Python, SQL, and Flask for the backend and HTML and CSS for the frontend. We built this out over the course of two months with different partners each time. It was a great learning experience!

  • Smiling School

    I was asked to recreate a website advertising a school for "smiling skills" that utilized implementations of vanilla JS, Bootstrap, Figma, HTML, and CSS. My codebase for this project was my own previous work, so I had to make sure to cross all the t's! I am pretty happy with how this turned out, and I loved feeling the satisfaction of finishing a full, multi-page site on my own.

  • Headphones

    This was my first full frontend project for my first trimester as a Fullstack web dev student at Holberton School, Tulsa. The tech stack used was Figma, HTML and CSS.
